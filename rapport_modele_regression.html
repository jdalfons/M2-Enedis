RAPPORT MODÈLE DE RÉGRESSION¶

1. Introduction¶

Dans ce rapport, nous utilisons un modèle d'arbre de régression pour prédire la consommation annuelle d'un logement. Cette consommation annuelle comprends les 5 usages que l'on a dans un logement, à savoir :

  • Eau chaude sanitaire (ECS)
  • Chauffage
  • Climatisation
  • Eclairage
  • Auxiliaires

La consommation annuelle de ces 5 usages est exprimé en kWhef/an.
Nous allons explorer les données, les prétraiter, entraîner le modèle et évaluer ses performances.

    ('imputer', SimpleImputer(strategy='most_frequent')),
    ('encoder', ce.OneHotEncoder(handle_unknown='ignore'))
])

# On définit le preprocessor
preprocessor = ColumnTransformer(
    transformers=[
        ('num', num_transformer, num_features),
        ('cat', cat_transformer, cat_features)
    ]
)

# On définit le pipeline
pipeline = Pipeline(steps=[
    ('preprocessor', preprocessor),
    ('regressor', DecisionTreeRegressor(max_depth=20, min_samples_leaf=10, min_samples_split=20))
])
